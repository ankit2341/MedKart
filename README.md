## About the project
E shopping web app for medicines and healthcare products. This is project is work in progress

## User features
- Login and signup
- Googlelogin
- Explore products
- Filter products
- Search products
- Book lab tests
- Profile
- Admin Panel
- Chatbot


## Admin features
- Update and delete users
- Update and delete products
- View orders
- View lab tests
